## About Me

I’m a junior software engineer and founder of **Domiverse Studio**, crafting immersive game worlds with clean, object‑oriented code. My work bridges gameplay programming, backend development, and low‑poly 3D art – all heavily inspired by classic PS1 aesthetics, dithering, and retro rendering constraints.

- **Currently building:** *Trashborn* (PSX‑style action roguelike) and *Backroad Outlaws* (PSX‑style multiplayer rally).
- **Learning:** Spring Boot, PostgreSQL, and system architecture for scalable applications.
- **Core skills:** C# (Unity), Network Programming (Mirror/Netcode for GameObjects), 3D Pipeline (Blender, sculpting, texture painting).
